# Tenzies
Tenzies is a game where you keep rolling dice until all of them are the name number.
You can hold a dice to preserve its number.
All dice must be held and must be the same number to win the game.

## Play
[https://rugvedb133.github.io/Tenzies-Game/](https://rugvedb133.github.io/Tenzies-Game/)

## Getting started
Install the dependencies and run the project
```
npm install
npm start
```
